Project System Engineering and Analysis - quizsysteem

Beschrijving:

Het hoofddoel van dit project was het digitaliseren van een quiz. Op het moment organiseerde deze vzw de quizmomenten op papier. De doelgroep voor deze quiz waren oudere mensen dus moest het enorm gebruiksvriendelijk zijn. Het quizsysteem moest aan bepaalde eisen voldoen. De quizmaster moest controle hebben over de snelheid van de vragen/rondes. De quizmaster moest ook een scorebord kunnen laten tonen dat telkens geüpdatet wordt. Ook was er een jury waar we rekening mee moesten houden. De jury moest de huidige score per team kunnen aanpassen en ook de juiste antwoorden in de verbetersleutel van te voren doorgeven. De verbetersleutel moest natuurlijk ook aangepast kunnen worden. De antwoorden van de teams moesten via een fysieke knoppendoos.

Wij hadden dit project aangepakt door een custom API te maken dat via webpagina’s de API kan aanspreken en zo data in en uit een database te halen. De Api was volledig in Python geschreven. Waar ik me vooral mee bezig heb gehouden was de front-end en de styling van de front-end. De front-end zijn de webpagina’s dat de Api gaat aanspreken door de interactie van de gebruiker. Dit hebben we gedaan aan de hand van REACT en ik heb deze pagina's stijl gegeven met CSS. Waar ik me ook mee bezig heb gehouden was de fysieke opstellen om de antwoorden door te geven. Dit hebben we gedaan door wat code te schrijven op een Raspberry Pi dat ook de API kan aanspreken. Aan deze Raspberry Pi waren knoppen met Ledjes aan verbonden.

Heel dit project werd uitgevoerd op een agile manier. Dit wil zeggen dat wij verschillende fases hebben doorlopen om vervolgens ons eindresultaat te krijgen. Tijdens deze fases heb ik enorm veel soft skills bijgeleerd en aangescherpt om dan uiteindelijk tot de proof of concept te komen.

Project IOT Essentials - weerstation

Beschrijving:

Dit was een van mijn eerste projecten binnen mijn opleiding in het hoger. Het project was volledig zelfstandig en het doel was om een weerstation te maken dat waardes via sensoren kan aflezen en vervolgens naar een Raspberry Pi doorstuurt. De 2 sensoren die bevestigd waren kunnen de temperatuur, hoogte en lichtlevel lezen. Deze waardes moesten vervolgens op een webpagina met grafieken getoond kunnen worden. dit project was volledig in C++ en PHP geschreven. De styling op de webpagina werd gedaan met CSS.

Aangezien er een uitgebreid document en een video over dit document en het feit dit één van mijn eerste projecten was, heb ik veel kunnen leren over het efficiënt documenteren en uitwerken van een volledig project. Dit was het eerste project waar ik een volledig afgewerkt product heb afgeleverd, dus zeker een project om te herinneren.
